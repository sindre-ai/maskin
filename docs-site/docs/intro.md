# Maskin

**The open-source AI agent workspace for product teams — no code required.**

Maskin is a workspace where AI agents run your product development process end-to-end. Humans set direction, agents execute. Think of it as a project management tool where AI agents are the primary users — they create insights, propose bets, break down tasks, and execute work autonomously.

## What makes Maskin different?

- **Agents are first-class citizens.** AI agents and humans share the same identity model, the same API, and the same permissions. Agents aren't plugins — they're the primary operators.
- **Everything is an API.** If you can see it in the UI, an agent can call it. If an agent can do it, a human can too. One interface for all.
- **Humans steer, not drive.** Agents run autonomously and inform humans what they did. Humans course-correct when needed — like a self-driving car where you can always grab the wheel.
- **Open source.** Apache 2.0 licensed. Run it yourself, extend it, contribute to it.

## How it works

1. **Insights** — Raw feedback flows in from anywhere (support tools, Slack, meetings). Agents read it all.
2. **Bets** — Agents find patterns, cluster related feedback, and propose experiments ("bets") the team should try.
3. **Tasks** — Agents break bets into concrete work items and execute them. No human needed to drag cards across a board.
4. **Feedback Loop** — Results feed back as new insights. The cycle continues.
