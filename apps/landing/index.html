How It Works

Three steps to autonomous agents

Go from zero to running agents in under 3 minutes. No complex setup required.

1

Create a workspace

Set up your workspace with the workflows and integrations your team uses. Connect Slack, GitHub, or any tool via MCP.

docker run -p 3000:3000 maskin/maskin
2

Define your strategy

Create bets (strategic goals) and let agents break them into tasks. Or bring your own tasks — agents work either way.

3

Agents execute

Agents pick up tasks, do the work in sandboxed containers, and report back. You review and steer — they handle the rest.
